celery-task-freq
================

Print execution frequencies by task using the celery log file.

Usage
-----

::

    $ cat /var/log/celery/w2.log | python celery-task-freq.py 
    07: ############
    08: ########################################################
    09: ###############################################
    10: ###############################################
    11: ##############################################
    12: ############################################
    13: ####
    14: ################
    15: ##
    16: ####
